一个
与你共同成长的 Agent。

它不是一个绑定在 IDE 上的编码助手，也不是围绕单一 API 的聊天机器人外壳。它是一个自主 Agent，运行在你的服务器上，记住所学，并且运行时间越长，能力越强。

$ curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

支持 Linux、macOS 和 WSL2 · 无需前置条件 · 自动安装所有组件

在 Nous Portal 上注册

60 秒内快速上手

1

安装

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

安装 uv、Python 3.11，克隆仓库，并完成所有设置。无需 sudo 权限。

2

配置

bash 
# 交互式设置向导
hermes setup

# 或选择你的模型
hermes model

连接到 Nous Portal (OAuth)、OpenRouter (API 密钥) 或你自己的端点。

3

开始聊天

bash 
hermes

就这样。功能完整的交互式 CLI，包含工具、记忆和技能。

4

迈向多平台 (可选)

bash 
# 交互式消息网关设置向导
hermes gateway setup

# 启动消息网关
hermes gateway

# 安装为系统服务
hermes gateway install

逐步连接 Telegram、Discord、Slack 或 WhatsApp。可作为 systemd 服务运行。

5

保持更新

bash 
hermes update

拉取最新更改并重新安装依赖项。随时运行以获取新功能和修复。

原生 Windows 支持目前处于极度实验阶段且不受支持。 请安装 WSL2 并在其中运行 Hermes Agent。

功能特性

与您同在

通过单一消息网关连接 Telegram、Discord、Slack、WhatsApp 和 CLI —— 在一个平台开始，在另一个平台继续。

运行越久，能力越强

持久化记忆和自动生成的技能 —— 它学习您的项目，并且永远不会忘记如何解决问题。

定时任务自动化

用于报告、备份和简报的自然语言定时任务调度 —— 通过消息网关无人值守运行。

委派与并行化

拥有独立会话、终端和 Python RPC 脚本的隔离子 Agent，用于构建零上下文成本流水线。

真正的沙盒化

五种后端 —— 本地、Docker、SSH、Singularity、Modal —— 提供容器加固和命名空间隔离。

完整的网页与浏览器控制

网页搜索、浏览器自动化、视觉、图像生成、文本转语音以及多模型推理。

工具

40+ 内置工具 — 网络搜索、终端、文件系统、浏览器自动化、视觉、图像生成、文本转语音、代码执行、子Agent委派、记忆、任务规划、定时任务调度、多模型推理等。

平台

Telegram、Discord、Slack、WhatsApp、Signal、Email 和 CLI — 全部通过单一消息网关连接。可连接至 Nous Portal、OpenRouter 或任何 OpenAI 兼容的 API。

执行环境

可在本地、Docker、SSH、Modal、Daytona 或 Singularity 中运行。通过只读根目录、降权能力和命名空间隔离实现容器加固。

技能

40+ 捆绑技能，涵盖 MLOps、GitHub 工作流、研究等领域。Agent 可即时创建新技能，并通过开放的 agentskills.io 格式分享。可从 ClawHub、 LobeHub 和 GitHub 安装社区技能。

研究

支持并行工作器和检查点的批量轨迹生成。集成 Atropos 进行 RL 训练。可导出至 ShareGPT，配合轨迹压缩进行微调。
